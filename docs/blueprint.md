# **App Name**: Digital Folio

## Core Features:

- Tabbed Navigation: Left-side navigation panel with collapsible/expandable tabs for different content categories.
- Rich Text Editor: WYSIWYG editor with formatting options (alignment, bold, etc.) for creating and editing text-based content.
- Code Viewer: Display code files with syntax highlighting for various languages (Python, Lua, HTML, CSS, JS, C#, MD).
- File Upload & Display: Upload and display various file types (text, code, Word documents) and folder structures (similar to GitHub).
- Content Grid: Display saved content within each tab as a grid of small preview tiles.
- Firestore Integration: Store and retrieve all information and project data using Cloud Firestore.
- Theme Toggle: Switch between light and dark UI themes in the settings.

## Style Guidelines:

- Primary color: Soft blue (#A7D1E7), conveying trustworthiness and competence, appropriate for a portfolio.
- Background color: Light grey (#F2F4F7) for a clean, modern backdrop, appropriate for both light and dark modes.
- Accent color: Deep blue (#3A5A7A) to provide contrast and highlight key UI elements.
- Font pairing: 'Space Grotesk' (sans-serif) for headings, 'Inter' (sans-serif) for body text.
- Use simple, outline-style icons for tab navigation and content actions.
- Left sidebar navigation with expandable sections.
- Use subtle transitions and animations when opening/closing tabs and displaying content.
